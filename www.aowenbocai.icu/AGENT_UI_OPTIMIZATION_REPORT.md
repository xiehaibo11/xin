#### 统计卡片
- **设计特色**: 左侧彩色边框标识

- **交互效果**: 
  - 行悬停: 背景色变化 + 轻微缩放

- **响应式**: 小屏幕下自动滚动

#### 按钮系统
- **统一样式**: 圆角、大写字母、适当间距
- **渐变背景**: 每种类型都有独特的渐变效果
- **交互反馈**: 
  - 悬停: 向上浮动 + 彩色阴影
  - 点击: 波纹扩散效果
- **图标集成**: 所有按钮都包含相关图标

#### 导航菜单
- **激活状态**: 左侧蓝色边框 + 背景渐变
- **悬停效果**: 向右滑动5px + 颜色变化
- **图标对齐**: 统一的图标宽度，保持整齐
